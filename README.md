# NestJS Product Service

This is a NestJS-based service that fetches product data from an external API and stores it in a Postgres database. The application is designed to ensure that the server retrieves and serves the latest product information. The service also provides public and private modules for managing and reporting product data, all of which are Dockerized using Docker Compose.

## Features

- **Cron Job**: Periodically fetches data from an external API and updates the Postgres database to ensure up-to-date product information.
- **Public Module**: 
  - Provides paginated product results.
  - Allows filtering by product name, category, and price range.
  - Enables product removal.
- **Private Module**:
  - Generates reports for:
    - Percentage of deleted products.
    - Percentage of non-deleted products between a specified `startDate` and `endDate`, and if products have a price (`hasPrice`).
    - Average price by product category and brand.

## Technologies

- **NestJS**: Framework for building scalable server-side applications.
- **Postgres**: Relational database for storing product information.
- **Docker Compose**: For containerizing the application and simplifying deployment.

## Installation

### Prerequisites

- [Docker](https://www.docker.com/get-started) installed on your machine.
- [pnpm](https://pnpm.io/) package manager (optional but recommended for package management).

## Project setup

```bash
$ pnpm install
```

## Compile and run the project


```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod

# Run with database
$ docker compose up

# Run only database
$ docker compose up db
```

## Run tests

```bash
# unit tests
$ pnpm run test

# test coverage
$ pnpm run test:cov
```
